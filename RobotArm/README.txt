
Python Drawing Arm G-code to Pulse Converter

Dependencies:
  pip install svgpathtools numpy

How to Use:
1. Put your 'drawing.svg' file in the same folder.
2. Run: python main.py
3. Output 'drawing.csv' will contain pulse1, pulse2, pen state

Drawing arm specs:
- Two-link arm (each 0.6 meters)
- Output pulses per radian: 1000
